Klein.php with Smarty template engine.

## Install

1) Install via composer.

```sh
composer create-project f3l1x/kleinsmarty:dev-master
```

2) Create app/cache folder.

3) Open KleinPHP-Smarty root folder and run **server.php**.

```php
php server.php
```
